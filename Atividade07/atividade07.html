Exercícios JavaScript - DOM e Eventos

1. O que é o DOM?

O DOM (Document Object Model) é a representação da estrutura de uma página HTML como um objeto em árvore, permitindo que linguagens como JavaScript acessem, manipulem e modifiquem os elementos da página dinamicamente.

2. Diferença entre getElementById e querySelector

getElementById: seleciona um elemento pelo seu ID.
querySelector: seleciona o primeiro elemento que corresponde a um seletor CSS (pode ser classe, ID, tag).

3. O que são eventos?

Eventos são ações que ocorrem na página (ex: clique, envio de formulário, digitação). Exemplos: click, mouseover, keydown.

4. Diferença entre innerText e innerHTML

innerText: retorna apenas o texto visível do elemento.
innerHTML: retorna o conteúdo HTML (incluindo tags) dentro do elemento.

5. Alterar estilo via JavaScript

Podemos usar a propriedade style do elemento. Ex: element.style.backgroundColor = "blue";
